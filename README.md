# Employee Management System

## Description

A GUI-based Employee Management System using Python, Tkinter, and MongoDB.

## Features

- User Login
- CRUD Operations
- Search
- Input Validation
- CSV Export
- MongoDB Integration

## Technologies

- Python
- Tkinter
- MongoDB
- PyMongo

## How to Run

1. Start MongoDB
2. Install requirements: pip install pymongo
3. Run: python main.py

## Scenario

Company employee record management
